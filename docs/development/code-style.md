# Code Style Guide

## Backend (TypeScript)

- Lint:

```bash
cd backend
npm run lint
npm run lint:fix
```

- Build:

```bash
npm run build
```

## Mobile (Dart/Flutter)

- Format:

```bash
cd mobile-app
dart format .
```

- Static analysis:

```bash
flutter analyze
```

## General

- Prefer small PRs with focused changes.
- Keep docs links relative and up-to-date.
