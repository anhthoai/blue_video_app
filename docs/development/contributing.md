## Adding/Updating docs

- Prefer updating an existing doc before creating a new one.
- When you do add a doc under `docs/`, ensure it’s linked from [`docs/README.md`](../README.md).
- For feature-specific long guides, it’s OK to keep them at repo root (many already exist); add a link in [`docs/reference-guides.md`](../reference-guides.md).
