Руководство по установке

How to install on iPhone

No App Store needed. Follow these steps in Safari — other browsers will not work.

  1. Open this page in Safari and tap Install on iPhone.
  2. Safari asks “This website is trying to download a configuration profile. Allow?” — tap Allow.
  3. “Profile Downloaded” appears at the top — tap Close.
  4. Go to Settings → General → VPN & Device Management.
  5. Tap the downloaded profile, then tap Install in the top-right corner.
  6. Enter your passcode and confirm. If it says “Unsigned Profile” that is normal — tap Install again.
  7. Return to the Home Screen. A Blue Video icon now appears — tap it to open the install page, then tap Open iPhone Install to install the app.
