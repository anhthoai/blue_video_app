First-time install help

How to install on iPhone

Users do not need the App Store first, but the install path is different from a normal public app install.

  1. Open this page in Safari on your iPhone and tap Open iPhone Install.
  2. If TestFlight opens, install TestFlight and accept the invite. If Safari shows an install prompt, approve it.
  3. After the app icon appears, open Settings > General > VPN & Device Management if iOS asks you to trust the developer.
  4. Return to the Home Screen and open Blue Video.
